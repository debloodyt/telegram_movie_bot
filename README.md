# Telegram Movie Bot

Проект: бот для Telegram, позволяющий искать фильмы и просматривать подборки на основе рейтинга.

##  Описание

Этот бот реализован на Python с использованием библиотеки aiogram и позволяет:
- искать фильмы по названию (/movie_search);
- получать топ фильмов по рейтингу (/movie_by_rating);
- получать справку по командам (/help);
- запускать стартовое приветствие (/start).

##  Структура проекта

- `main.py` — точка входа, инициализация бота, диспетчер команд.
- `config.py` — хранит конфигурацию и ключи.
- `pagination.py` — логика пагинации результатов поиска.
- `handlers/` — обработчики команд:
  - `start.py`
  - `help.py`
  - `movie_search.py`
  - `movie_by_rating.py`
- `requirements.txt` — зависимости проекта.
- `readme.md` — документация.

##  Установка

1. Клонировать репозиторий:
   git clone https://github.com/debloodyt/telegram_movie_bot.git
   cd telegram_movie_bot

2. Создать и активировать виртуальное окружение:
   python -m venv venv
   .\venv\Scripts\Activate.ps1    # PowerShell
   # или .\venv\Scripts\activate   для cmd

3. Установить зависимости:
   pip install -r requirements.txt

##  Настройка

1. Откройте config.py.
2. Установите ваш TELEGRAM_TOKEN (токен бота от @BotFather).
3. При необходимости настройте другие константы (например, API URL, лимиты).

##  Запуск бота
python main.py

Бот будет доступен в Telegram по вашему токену.

##  Использование

- /start  приветствие и краткое описание функционала
- /help  список команд
- /movie_search поиск фильмов по ключевым словам
- /movie_by_rating  вывод списка фильмов по рейтингу (с возможностью пагинации)
